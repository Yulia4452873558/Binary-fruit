# 🥷 Ninja Aurora

**Slice 'em all!**  
Ninja Aurora — аркадная игра в жанре «слайсер» для операционной системы «Аврора». Режьте фрукты, уворачивайтесь от бомб и ставьте рекорды. Разработано на Godot Engine.

## 🎮 Готовые сборки

В корне репозитория находятся готовые артефакты релиза (после сборки):
* `NinjaAurora.apk` — сборка для Android.
* `[...].rpm` — сборка для ОС «Аврора».

**Демо-релиз отмечен тегом:** `demo-2026-05-18`

## 🚀 Быстрый старт

### Запуск из исходников
1. Установите [Godot Engine](https://godotengine.org/) версии 4.x.
2. Клонируйте репозиторий: `git clone https://github.com/top-it-090304/Binary-fruit.git`
3. Откройте проект в Godot, выбрав файл `project.godot`.
4. Запустите основную сцену (`game.tscn` или `main_menu.tscn`).

### Установка на Android
1. Скачайте файл `NinjaAurora.apk` из раздела [Releases] или корня репозитория.
2. Разрешите установку из неизвестных источников на вашем устройстве.
3. Установите APK и запустите игру.

### Установка на ОС «Аврора»
1. Скачайте RPM-файл.
2. Установите его стандартным способом для вашего устройства Аврора или через SDK.

## ✂️ Геймплей
*   Режьте фрукты (апельсины, арбузы и другие).
*   Избегайте бомб — они отнимают жизнь.
*   Набирайте комбо для получения множителя очков.
*   Интуитивное сенсорное управление.
